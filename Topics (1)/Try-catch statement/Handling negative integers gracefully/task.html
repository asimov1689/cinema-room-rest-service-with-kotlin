Write a program that prompts the user to enter a non-negative integer. If the user enters a negative integer, the program should catch the exception and print an error message. Otherwise, the program should print the square of the input integer.


Sample Input:
5

Sample Output:
25


Sample Input:
-3

Sample Output:
Invalid input! Please enter a non-negative integer.



Memory limit: 256 MB
Time limit: 8 seconds
